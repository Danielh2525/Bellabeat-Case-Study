# Bellabeat-Case-Study - Analytics for Decision Making

Project Overview:


This project analyzes smart-device usage data to identify behavioral patterns related to physical activity, sleep, and lifestyle habits. The goal is to translate raw user data into actionable insights that can inform Bellabeatâ€™s marketing and product strategy.

This case study was completed as part of the Google Data Analytics Capstone, with a strong emphasis on structured analysis, clear reasoning, and decision-oriented outcomes.


# Business Objectivee

Bellabeat aims to better understand how users engage with smart health devices in order to:

Improve customer engagement

Tailor marketing strategies

Prioritize product and feature development

The key analytical question:

How do user activity and lifestyle patterns relate to engagement, and how can these insights guide business decisions?


# Data & Tools

Tools:

Python (Pandas, NumPy, Matplotlib, Seaborn)

Jupyter Notebook

Techniques:

Data cleaning and preprocessing

Exploratory Data Analysis (EDA)
